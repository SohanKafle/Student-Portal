# 🎓 NextGen Student Portal

A sleek, single-page **Student Registration Portal & Directory** featuring a futuristic Glassmorphism UI, real-time ID generation, and local data persistence.

Developed with ❤️ by **Sohan Kafle** 🌐 [Website](https://sohankafle.com.np) | 🐙 [GitHub](https://github.com/SohanKafle) | 💼 [LinkedIn](https://www.linkedin.com/in/sohan-kafle/)

---

### ✨ Features
* **🪞 Glassmorphism UI:** Translucent layers, smooth blurs, and vibrant mesh gradients.
* **⚡ Live ID Preview:** Real-time digital badge updates with dynamic Dicebear API avatars.
* **🔐 Admin Directory:** Secure administrative dashboard with metric tracking and profile management.
* **💾 Persistent Data:** Save profiles directly to browser `localStorage` (no backend needed).
* **📱 Responsive Design:** Flawless layout across mobile, tablet, and desktop viewports.
